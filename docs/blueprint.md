# **App Name**: Excellere Revive

## Core Features:

- Module Access: Maintain existing module access and security structure.
- Iconography Navigation: Replace text-based navigation with intuitive iconography for a modern user experience.
- Collapsible Menu: Implement a space-saving collapsible menu for improved interface cleanliness.
- Data Persistence: All app data stored to Firestore, just as before.

## Style Guidelines:

- Primary color: Soft green (#8FBC8F) to maintain a connection to the original branding.
- Background color: Light grey (#F0F0F0) for a modern and clean backdrop.
- Accent color: Dark grey (#586859) for contrast and focus on key elements.
- Font: 'Inter' sans-serif font; modern look for body and headings.
- Use minimalist, line-style icons throughout the interface for navigation and actions.
- Employ a grid-based layout to ensure responsive design and consistent element spacing.
- Add subtle animations to menu transitions and button interactions for a smoother feel.
